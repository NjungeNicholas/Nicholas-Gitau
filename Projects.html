Impulse

In Progress

A fitness-powered screen-time control app for Android. Impulse lets you earn phone usage time by completing physical exercise — detected on-device via sensors. Locked apps can only be accessed after a workout is logged, creating a healthy feedback loop between movement and screen time.

Built for: Android users who want to reduce mindless scrolling and build an exercise habit.

Jobshield

Done

A web extension that uses pattern-matching algorithms to detect and block fraudulent job ads on Kenyan job boards. Jobshield scans job listings for red flags like unrealistic salaries, vague descriptions, and suspicious contact info — helping job seekers avoid scams and find legitimate opportunities.

Built for: Job seekers in Kenya looking for a safer way to find legitimate employment opportunities.

QuickGig

Done

A web application that connects Kenyan gig workers with short-term job opportunities. QuickGig allows users to create profiles showcasing their skills and availability, then matches them with local gigs in areas like delivery, tutoring, and handyman services. The platform includes a secure payment system and user reviews to build trust between workers and clients.
